#### 介绍
- 项目需求：
    - 控制台实现2048游戏。
    - 如果想要了解这个游戏的话：https://play2048.co/ 
- 项目思想和逻辑：
    - 由于该项目中追求的是尽量少的代码，所以阅读体验估计会很差
    - 但是**核心思想表现的很清楚**
- 关键环境：
    - python 3.65 及标准库即可
- 命令行测试：
    - python tzfe.py即可
- 项目补充：
    - 清屏命令使用的是win下的 cls，如果使用linux的话请换成 clear
    - 关于程序设计中的求和操作中的算法应该可以优化，但是我没优化，这主要是我自己又去玩了一遍游戏，发现自己设计的和原版有出入，又新添的代码。
